# Lucas Zeiger — Portfolio

Simple personal site built with Vite + React + TypeScript and Tailwind (CDN). Routing uses `HashRouter` for GitHub Pages.

## Run Locally
- Requires Node.js
- Install: `npm install`
- Dev server: `npm run dev` → open the printed URL

## Build
- Production build: `npm run build`
- Preview the build: `npm run preview`

## Deploy (GitHub Pages)
- Repo name: `LucasZeiger.github.io`
- Publish: `npm run deploy` (builds and pushes `dist` to `gh-pages`)
- GitHub Settings → Pages → Source: branch `gh-pages` (root) → site at https://LucasZeiger.github.io/
